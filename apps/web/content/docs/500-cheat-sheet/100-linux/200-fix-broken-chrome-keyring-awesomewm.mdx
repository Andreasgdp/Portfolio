# Issue

When using AwesomeWM, the Chrome keyring can become broken unable to save chrome passwords.

# Solution

1. Close Chrome.
2. `cd ~/.config && mv google-chrome google-chrome_old`.
3. Start Chrome.
4. Sign in to Chrome - This will prompt you like a new install.
5. Check if sync is activated
